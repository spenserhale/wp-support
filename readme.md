## WP Support

![Screen Shot][product-screenshot]

WP Support is a PHP Library that contains utility classes for modern WordPress development.

<!-- GETTING STARTED -->

## Getting Started

### Installation

   ```sh
   composer require spenserhale/wp-support
   ```

## Tests

Then run the tests:
   ```sh
    composer test
   ```
